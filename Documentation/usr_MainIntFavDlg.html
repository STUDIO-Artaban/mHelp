Interface Principale: Macros Favorites

L'interface "Tâches Favorites" appelée via le bouton "Tâche Favorite" de l'onglet "Fichier(s) DynHelp" (voir Interface Principale: Onglet "Fichier(s) DynHelp"), vous permet de charger et de supprimer les macros favorites enregistrées dans le fichier DynHelp sélectionné.


Evènement Description de la macro favorite enregistrée dans le fichier DynHelp.
Répétition Nombre de répétitions de la macro favorite enregistrée dans le fichier DynHelp.
Rang Position dans la liste de la macro favorite enregistrée dans le fichier DynHelp.
Liste de macros favorites Liste des macros favorites enregistrées dans le fichier DynHelp.
Fichier DynHelp Nom et chemin complet du fichier DynHelp où sont enregistrées les macros favorites.
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Détruire Permet de détruire du fichier DynHelp la macro favorite sélectionnée.
Ajouter Fav. Permet de charger la macro favorite sélectionnée dans la liste des macros favorites de l'onglet "Favorites" (voir Interface Principale: Onglet "Favorites").
Fermer Permet de fermer l'interface.

Remarque

L'opération de sauvegarde des macros favorites dans le fichier DynHelp, peut être réalisée à partir de l'onglet "Favorites" via le bouton "Enregistrer" (voir Interface Principale: Onglet "Favorites").

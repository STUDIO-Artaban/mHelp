Interface d'Enregistrement: Paramètre(s) de la Macro

L'étape "Paramètre(s) de la Macro" de l'interface d'enregistrement, vous permet de définir la description et la limite de chacun des paramètres de la macro paramétrable, avant son enregistrement. Cette étape vous permet également de choisir quel paramètre vous souhaitez laisser à l'utilisateur la possibilité d'en éditer la valeur.


Type Type de contrôle paramétrable.
Etat/Sélection(s) Etat du contrôle paramétrable (valeur du paramètre).
Application Nom de l'application dans lequel est intervenue l'action sur le contrôle paramétrable (voir Remarques).
Visible Détermine si le paramètre sera visible ou non dans la liste des paramètres, une fois la macro paramétrable enregistrée.
Rang Position du contrôle dans la liste des contrôles (voir Interface d'Enregistrement: Liste des Contrôles).
Liste des contrôles Liste des contrôles paramétrables.
Configurer Permet de configurer le contrôle sélectionné, et de définir la description et la limite du paramètre associé à ce contrôle (voir Interface de Configuration: Onglet "Commun").
Cacher / Réafficher Permet de retirer / laisser le paramètre du contrôle sélectionné, dans la liste des paramètres de la future macro paramétrable. De cette manière, il vous est possible de choisir entre laisser ou non l'utilisateur la possibilité de configurer l'état de ce contrôle (valeur du paramètre), avant de lancer cette macro (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
‹ Précédent Permet de revenir à l'étape précédente : Description de la Macro (voir Interface d'Enregistrement: Description de la Macro).
Enregistrer Permet d'enregistrer la macro paramétrable ainsi configurée, avant de revenir à la première étape (voir Interface d'Enregistrement: Liste des Contrôles), où vous avez la possibilité de reprendre la création de la macro ou de l'enregistrer à nouveau dans le fichier DynHelp de votre choix.
Terminer Permet de terminer le processus de création de macros.

Remarques

L'intérêt d'afficher le nom de l'application dans laquelle a été réalisée l'action sur un contrôle, est lié à une des particularités du système DynHelp qui vous permet de créer des macros passant d'une application à une autre (applications utilisant le système DynHelp).

Configurer la valeur du paramètre associé à l'état d'un contrôle lorsque celui-ci est cacher de la liste des paramètres (voir Cacher / Afficher), revient à déterminer l'état dans lequel il sera positionné, à chaque lancement de la future macro paramétrable (comme pour une macro non paramétrable).

Interface de Configuration: Onglet "Etat" de Type Chaîne de caractères

L'onglet "Etat" de l'interface de configuration, vous permet de modifier la valeur du paramètre associé à l'état d'un contrôle, initialement fixée durant la création de la macro. Cette interface dépend du type de contrôle et donc du type de paramètre qui y est associé (voir Description des Contrôles et des Types de Paramètre). Dans les exemples qui suivent il s'agit d'un paramètre de type Chaîne de caractères (S).

Contrôles de type: ComboBox, ListBox (simple sélection), ListView (simple sélection), TabCtrl


Commun Onglet "Commun" permettant de configurer le comportement à suivre dans le cas où le contrôle en cours de configuration, serait invisible ou inactif durant (voir Interface de Configuration: Onglet "Commun").
A Propos Onglet contenant des informations sur le contrôle sélectionné.
Chaîne de caractères à sélectionner Permet de définir la chaîne de caractères à sélectionner (valeur du paramètre).
Description Permet de définir la description du paramètre (voir Remarques).
Type Affiche le type de paramètre (voir Description des Contrôles et des Types de Paramètre).
Limite Permet de définir la limite du paramètre (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Appliquer Permet d'appliquer les changements configurés pour le contrôle sélectionné.
Fermer Permet de fermer l'interface.

Il est conseillé de cacher de la liste des paramètres ceux qui sont liés à un TabCtrl (voir Interface d'Enregistrement: Paramètre de la Macro), car le bon déroulement de la suite des actions contenues dans cette future macro paramétrable, peut dépendre de l'onglet sélectionné.

Contrôle de type: EditBox


Contenu de la zone d'édition / Description du contenu Permet de définir la chaîne de caractères à placer dans la zone d'édition (valeur du paramètre) / Permet de définir la description du contenu de la zone d'édition, avant que l'utilisateur ne fasse sa saisie.
Contenu fixe Option permettant de prédéfinir le contenu de la zone d'édition.
Saisi par l'utilisateur Option permettant de laisser l'utilisateur la possibilité de saisir lui-même le contenu de la zone d'édition, durant le lancement de la macro. Une fois le contenu défini, l'utilisateur n'a plus qu'à taper la touche "Entrée" pour que le processus de lancement reprenne.

L'option permettant à l'utilisateur de saisir lui-même le contenu de la zone d'édition, dépend de la manière dont est gérée cette zone et cela a pour effet de supprimer de la liste des paramètres, le paramètre qui est associé à ce contrôle. C'est pourquoi, lorsque vous souhaitez configurer ce paramètre via la liste des paramètres, cette option est désactivée.

Contrôle de type: Change d'Application

Ceci n'est pas un contrôle mais une action, représentant un changement d'application.


Ligne de commandes Permet de définir la ligne de commandes de l'exécutable (valeur du paramètre).

Ce type de "contrôle" ne comporte pas d'état lorsqu'il ne s'agit pas de lancer une nouvelle application, mais d'activer une application qui a déjà été lancé par le processus DynHelp.

Il est conseillé de cacher ce paramètre de la liste des paramètres (voir Interface d'Enregistrement: Paramètres de la Macro), car de sa valeur peut dépendre le fonctionnement de l'application, risquant ainsi de perturber le bon déroulement de la suite des actions contenues dans la future macro paramétrable.

Remarques

Les zones d'édition permettant de définir la description du paramètre et de ses limites, ne peuvent être éditées que dans le cas où vous souhaitez créer une macro paramétrable, et qu'il s'agit là de la configuration d'un de ses paramètres (voir Interface d'Enregistrement: Paramètres de la Macro).

S'il s'agit de la configuration d'un des paramètres de la future macro paramétrable, modifier l'état du contrôle revient à configurer la valeur par défaut du paramètre qui y est associé.

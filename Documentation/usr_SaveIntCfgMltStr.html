Interface de Configuration: Onglet "Etat" de Type Liste de Chaînes de caractères

L'onglet "Etat" de l'interface de configuration, vous permet de modifier la valeur du paramètre associé à l'état d'un contrôle, initialement fixée durant la création de la macro. Cette interface dépend du type de contrôle et donc du type de paramètre qui y est associé (voir Description des Contrôles et des Types de Paramètre). Dans l'exemple qui suit il s'agit d'un paramètre de type Liste de Chaînes de caractères (LS).

Contrôles de type: ListBox (multiple sélections), ListView (multiple sélections), TreeView


Commun Onglet "Commun" permettant de configurer le comportement à suivre dans le cas où le contrôle en cours de configuration, serait invisible ou inactif durant (voir Interface de Configuration: Onglet "Commun").
A Propos Onglet contenant des informations sur le contrôle sélectionné.
Nouvelle chaîne de caractères Permet de définir une nouvelle chaîne de caractères à sélectionner.
Liste de chaînes de caractères Liste des chaînes de caractères à sélectionner.
Ajouter Permet d'ajouter dans la liste des chaînes de caractères à sélectionner, la nouvelle chaîne de caractères définie ci-dessus (Nouvelle chaîne de caractères).
Retirer Permet de retirer de la liste des chaînes de caractères à sélectionner, la valeur sélectionnée dans cette même liste.
Description Permet de définir la description du paramètre (voir Remarques).
Type Affiche le type de paramètre (voir Description des Contrôles et des Types de Paramètre).
Limite Permet de définir la limite du paramètre (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Appliquer Permet d'appliquer les changements configurés pour le contrôle sélectionné.
Fermer Permet de fermer l'interface.

Pour un contrôle de type TreeView, l'ordre dans laquelle sont placées les chaînes de caractères à sélectionner, a son importance. En effet, il s'agit là non pas de sélectionner chaque chaîne de caractères, mais de parcourir l'arborescence du TreeView, de la racine (première chaîne de caractères de la liste), au dernier élément de l'arborescence à sélectionner (dernière chaîne de caractères de la liste).

Remarques

Les zones d'édition permettant de définir la description du paramètre et de ses limites, ne peuvent être éditées que dans le cas où vous souhaitez créer une macro paramétrable, et qu'il s'agit là de la configuration d'un de ses paramètres (voir Interface d'Enregistrement: Paramètres de la Macro).

S'il s'agit de la configuration d'un des paramètres de la future macro paramétrable, modifier l'état du contrôle revient à configurer la valeur par défaut du paramètre qui y est associé.

Description de l'Interface d'Evolution des Paramètres

Cette interface est appelée via le bouton "..." dans l'onglet "Sommaire" de l'interface principale (voir Interface Principale: Onglet "Sommaire"), dès lors qu'il y a une répétition et que vous avez sélectionné un paramètre. Elle permet de réaliser l'opération suivante :

Configurer chaque valeur du paramètre associée à l'état d'un contrôle, pour chaque répétition.

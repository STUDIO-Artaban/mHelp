Description des Eléments de l'Arborescence

Voici la description des éléments contenus dans la description des macros sous forme d'arborescence :

Sujet du fichier DynHelp ouvert.
Partie de la description d'une ou plusieurs macros.
Fin de la description d'une macro paramétrable.
Fin de la description d'une macro non paramétrable.
Paramètre d'une macro paramétrable.
Paramètre d'une macro paramétrable configuré pour évoluer suivant la répétition (voir Interface Principale: Evolution des Paramètres).

Remarque

Les paramètres d'une macro paramétrable peuvent être éditer par l'utilisateur, en double cliquant sur l'icone d'un des paramètres.

Interface Principale: Evolution des Paramètres

L'interface "Evolution du Paramètre" vous permet de configurer les valeurs du paramètre sélectionné, suivant la répétition. Cette interface est appelée via le bouton "...", qui s'affiche à la sélection d'un paramètre (voir Interface Principale: Onglet "Sommaire"), dès lors qu'il y a une répétition.


Type Type de paramètre.
Limite Limite du paramètre.
Description Description du paramètre.
Type d'évolution Liste des outils d'évolution permettant de modifier de manière automatique les valeurs du paramètre.
Valeur associée Zone d'édition utilisée par certains outils d'évolution comme par exemple l'outil "Incrémenter", permettant de fixer la valeur d'incrémentation (voir Remarques).
Appliquer Permet d'appliquer le changement des valeurs du paramètre suivant le type d'outils d'évolution sélectionné.
Valider Permet de valider le changement des valeurs du paramètre suivant la répétition.
Retirer Permet de retirer le changement des valeurs du paramètre, conservant alors la première valeur du paramètre durant toutes les répétitions.
Annuler Permet d'annuler le changement des valeurs du paramètre suivant la répétition.
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Valeur Valeur du paramètre suivant la répétition.
Rang Index de la répétition.
Liste des valeurs Liste des valeurs du paramètre suivant l'index de la répétition (Rang).

Remarques

Les outils d'évolution sont utiles lorsqu'il y a beaucoup de répétitions, et que vous avez à effectuer certains changements spécifiques sur les valeurs du paramètre. Voici la description des différents types d'outils existants, pour faire évoluer le paramètre tout au long des répétitions:

Copier la 1° sel.
Ce type d'évolution vous permet, via le bouton "Appliquer", de copier la première valeur sélectionnée dans les valeurs sélectionnées qui suivent. Dans l'exemple plus haut, cela aurait pour effet de copier la valeur "Introduction.doc", dans celles contenant les valeurs "Projet.rtf", "CV.doc" et "Document1.doc".
Copier/Coller
Le type d'évolution "Copier/Coller" permet dans un premier temps, de copier les valeurs sélectionnées, pour ensuite les coller dans les nouvelles valeurs sélectionnées. Si, lors d'un collage, le nombre de nouvelles valeurs sélectionnées est supérieur au nombre de valeurs copiées, alors ces dernières seront répétées autant de fois qu'il est nécessaire.
Incrémenter
Pour les paramètres de type Entier ou Booléen, le type d'évolution "Incrémenter" permet d'incrémenter les valeurs sélectionnées, suivant la valeur associée.
Pour les paramètres de type Chaîne de caractères, cela permet d'ajouter et d'incrémenter à la fin de chaque valeurs sélectionnées, un nombre convertie en chaîne de caractères et s'incrémentant de un. L'option "Ext." permet d'adopter un comportement spécifique pour les valeurs représentant un nom de fichier, avec son extension (comme dans l'exemple plus haut). Cela a pour effet d'ajouter et d'incrémenter la valeur associée convertie en chaîne de caractères, non pas en fin des paramètres sélectionnées, mais juste avant l'extension du fichier, autrement dit, dans le nom du fichier.
Décrémenter
Ce type d'évolution est réservé aux paramètres de type Entier ou Booléen, et permet de décrémenter les valeurs sélectionnées, suivant la valeur associée.
Changer d'état
Ce type d'évolution est réservé aux paramètres de type Booléen, et permet de changer d'état toutes les énièmes sélections, définies dans la valeur associée ("Toutes les :").

Une fois les valeurs du paramètre configurés pour évoluer suivant la répétition, l'icône du paramètre change de l'image d'un document rouge à l'image d'un document vert (voir Description des Eléments de l'Arborescence).

Interface d'Enregistrement: Liste des Contrôles

La "Liste des contrôles" de l'interface d'enregistrement, vous permet de visualiser et de configurer les contrôles sur lesquels vous êtes intervenus, durant la création de votre macro.


Type Type de contrôle.
Etat/Sélection(s) Etat du contrôle dans le cas ou il s'agit d'un contrôle paramétrable (voir Description des Contrôles et des Types de Paramètre).
Application Nom de l'application dans lequel est intervenue l'action sur le contrôle (voir Remarques).
Rang Position du contrôle dans la liste des contrôles.
Liste des Contrôles Liste des contrôles sur lesquels vous êtes intervenus, durant la création de votre macro.
Configurer Permet de configurer le contrôle sélectionné (voir Interface de Configuration: Onglet "Commun").
Déplacer /\ Permet de déplacer le contrôle sélectionné, afin que l'action sur ce contrôle soit réalisée avant l'action sur le contrôle qui le précède (voir Remarques).
Déplacer \/ Permet de déplacer le contrôle sélectionné, afin que l'action sur ce contrôle soit réalisée après l'action sur le contrôle qui le suivant (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
‹ Précédent Permet de reprendre la création de la macro, là où vous vous êtes arrêté (voir Description de l'Interface d'Enregistrement de Macros).
Suivant › Permet de passer à l'étape suivante (voir Interface d'Enregistrement: Fichier d'Enregistrement).
Terminer Permet de terminer le processus de création de macros.

Remarques

L'intérêt d'afficher le nom de l'application dans laquelle a été réalisée l'action sur un contrôle, est lié à une des particularités du système DynHelp, qui vous permet de créer des macros passant d'une application à une autre (applications utilisant le système DynHelp).

La possibilité d'utiliser les outils de déplacement "Déplacer /\" et "Déplacer \/", dépends de la fenêtre active à ce moment là. En effet, vous ne pouvez pas réaliser une action sur un contrôle lorsque la fenêtre dans laquelle il se trouve n'est plus, ou n'est pas encore ouverte.

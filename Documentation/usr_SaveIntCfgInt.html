Interface de Configuration: Onglet "Etat" de Type Entier

L'onglet "Etat" de l'interface de configuration, vous permet de modifier la valeur du paramètre associé à l'état d'un contrôle, initialement fixée durant la création de la macro. Cette interface dépend du type de contrôle et donc du type de paramètre qui y est associé (voir Description des Contrôles et des Types de Paramètre). Dans les exemples qui suivent il s'agit d'un paramètre de type Entier (I).

Contrôle de type: Slider/TrackBar


Commun Onglet "Commun" permettant de configurer le comportement à suivre dans le cas où le contrôle en cours de configuration, serait invisible ou inactif durant (voir Interface de Configuration: Onglet "Commun").
A Propos Onglet contenant des informations sur le contrôle sélectionné.
Position du Slider Permet de modifier l'état initial du contrôle, de manière à changer la position de ce Slider lors du lancement de la macro.
Valeur de la position Valeur du paramètre.
Description Permet de définir la description du paramètre (voir Remarques).
Type Affiche le type de paramètre (voir Description des Contrôles et des Types de Paramètre).
Limite Permet de définir la limite du paramètre, dont la valeur par défaut est déterminée par les propriétés du Slider (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Appliquer Permet d'appliquer les changements configurés pour le contrôle sélectionné.
Fermer Permet de fermer l'interface.

Contrôles de type: RadioBox, ToolBar radiobox

Lorsque l'on agit sur un contrôle de type RadioBox, c'est pour faire un choix parmi d'autres RadioBoxs. Un RadioBox seul n'a donc pas d'état en soi. C'est pourquoi, cette interface vous permet de regrouper plusieurs de ces contrôles afin d'en extraire un paramètre, correspondant au choix du RadioBox que vous souhaitez sélectionner.

Pour ce faire, vous devez lors de la création de la macro, sélectionner l'un après l'autre tous les RadioBoxs qui sont groupés et qui forment ainsi un choix d'options, puis, via l'interface d'enregistrement, configurer l'un de ces contrôles en s'assurant que les autres RadioBoxs qui le précèdent ou qui le suivent dans la liste des contrôles (voir Interface d'Enregistrement: Liste des Contrôles), soient bien présents.


Radioboxs sélectionnés Noms des RadioBoxs sélectionnés.
Rang Rang du RadioBox dans la liste des contrôles (voir Interface d'Enregistrement: Liste des Contrôles).
Liste des RadioBoxs Liste des RadioBoxs qui précèdent ou qui suivent le contrôle à configurer, et qui représente les différents choix possibles.
Lier Permet de regrouper les RadioBoxs sélectionnés dans la liste des RadioBoxs.
Délier Permet de séparer les RadioBoxs regroupés et de supprimer ainsi le paramètre associé.
Rang du Radiobox Permet de choisir le RadioBox à sélectionner suivant sa position dans la liste des RadioBoxs regroupés (valeur du paramètre).
Limite Permet de définir la limite du paramètre, dont la valeur par défaut est déterminée par le nombre de RadioBoxs regroupés (voir Remarques).

Une fois liées, vous pouvez alors constater qu'il n'y a plus qu'un seul de ces contrôles dans la liste des contrôles, avec pour état le rang du RadioBox à sélectionner parmi les autres.

Cette opération ne peut être réalisée qu'à partir de la liste des contrôles sur lequel l'utilisateur a agit (voir Interface d'Enregistrement: Liste des Contrôles).

Remarques

Les zones d'édition permettant de définir la description du paramètre et de ses limites, ne peuvent être éditées que dans le cas où vous souhaitez créer une macro paramétrable, et qu'il s'agit là de la configuration d'un de ses paramètres (voir Interface d'Enregistrement: Paramètres de la Macro).

S'il s'agit de la configuration d'un des paramètres de la future macro paramétrable, modifier l'état du contrôle revient à configurer la valeur par défaut du paramètre qui y est associé.

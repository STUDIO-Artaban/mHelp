Interface d'Enregistrement: Description de la Macro

L'étape "Description de la Macro" de l'interface d'enregistrement, vous permet de définir la description de la macro avant de l'enregistrer dans le fichier DynHelp de votre choix. Cette description se présente sous la forme d'une arborescence (voir Description des Eléments de l'Arborescence).


Liste des macros Description des macros contenues dans le fichier dans lequel vous souhaitez ajouter celle en cours de création.
Répertoire Sélectionne l'élément de l'arborescence de type répertoire (voir Description des Eléments de l'Arborescence).
Document Sélectionne l'élément de l'arborescence de type document (voir Description des Eléments de l'Arborescence).
Paramètre(s) Sélectionne l'élément de l'arborescence de type paramètre (voir Description des Eléments de l'Arborescence).
Ajouter Permet d'ajouter à l'arborescence l'élément sélectionné.
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
‹ Précédent Permet de revenir à l'étape précédente: Fichier d'Enregistrement (voir Interface d'Enregistrement: Fichier d'Enregistrement).
Enregistrer / Suivant › Permet d'enregistrer la macro non paramétrable ainsi décrite / Permet de passer à l'étape suivante : Paramètre(s) de la Macro (voir Interface d'Enregistrement: Paramètre(s) de la Macro), dans le cas où il s'agit d'une macro paramétrable.
Terminer Permet de terminer le processus de création de macros.

Remarque

Revenir à l'étape précédente revient à annuler toutes modifications apportées à l'arborescence du fichier, dans lequel vous souhaitiez enregistrer la nouvelle macro.

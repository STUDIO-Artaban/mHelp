Description de l'Interface de Configuration des Contrôles

L'interface de configuration d'un contrôle est appelée au déclenchement du bouton "Configurer", de l'interface d'enregistrement (voir Interface d'Enregistrement: Liste des Contrôles et Interface d'Enregistrement: Paramètres de la Macro). Elle vous permet de réaliser les tâches décrites ci-dessous :

Choisir le comportement à suivre dans le cas où le contrôle serait invisible ou inactif, et de définir le message qui sera affiché dans cette éventualité.
Configurer la valeur du paramètre du contrôle, s'il s'agit d'un contrôle paramétrable.
Définir la description et la limite du paramètre de ce contrôle, pour une macro paramétrable.

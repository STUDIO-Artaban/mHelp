Description de l'Interface d'Enregistrement de Macros

L'interface d'enregistrement de macros intervient lorsque vous souhaitez enregistrer la macro que vous venez de créer, en validant l'interface de création de macro, décrite ci-dessous:


Annuler Annule la création d'une nouvelle macro.
Enregistrer Lance l'interface d'enregistrement de macros pour sauvegarder la macro ainsi créée.

L'interface d'enregistrement vous permet d'enregistrer votre nouvelle macro dans le fichier DynHelp de votre, après avoir configurer toutes les informations liées a cette macro, comme sa description et la description de ses paramètres (dans la cas d'une macro paramétrable).

Remarque

En cas d'erreur durant la création d'une macro, veuillez lancer l'application "DhpDgnst.exe" afin de réinitialiser l'état du système DynHelp avant de pouvoir le réutiliser (voir Description de l'Outil "DhpDgnst.exe").

Description de l'Interface des Macros Favorites

Cette interface est appelée via le bouton "Tâche Favorite" dans l'onglet "Fichier(s) DynHelp" de l'interface principale (voir Interface Principale: Onglet "Fichier(s) DynHelp"). Elle permet de réaliser les opérations suivantes :

Chargement d'une macro favorite stockée dans le fichier DynHelp sélectionné.
Suppression d'une macro favorite stockée dans le fichier DynHelp sélectionné.

DynHelp: Guide Utilisateur V1.21
Interface Principale: Onglet "Fichier(s) DynHelp"

L'onglet "Fichier(s) DynHelp" de l'interface principale, vous permet d'ouvrir et de fermer le ou les fichiers DynHelp, contenant les macros que vous souhaitez utiliser. Il vous assure également la gestion des macros favorites enregistrées dans ces mêmes fichiers.


Sommaire Onglet assurant l'affichage et la sélection des macros, contenues dans les différents fichiers DynHelp ouverts.
Configuration Onglet permettant de configurer certaines options, comme notamment le nombre de répétitions.
Rechercher Onglet destiné à rechercher une macro dans tous les fichiers DynHelp ouverts.
Index Onglet assurant l'affichage de la liste de tous les mots clés, contenus dans les différents fichiers DynHelp ouverts.
Favorites Onglet permettant de gérer les macros favorites et le déclenchement automatique.
A Propos Onglet contenant des informations sur le kit DynHelp.
Sujet Sujet/Titre du fichier DynHelp.
Chemin du fichier Nom et chemin complet du fichier DynHelp.
Fichier(s) DynHelp Liste des fichiers DynHelp ouverts.
Ouvrir Permet d'ouvrir un nouveau fichier DynHelp.
Retirer Permet de fermer le fichier DynHelp sélectionné.
Tâche Favorite Permet de charger ou de détruire les macros favorites enregistrées dans le fichier DynHelp sélectionné (voir Interface Principale: Macros Favorites).
Statut Statut du fonctionnement actuel.
Evènement sélectionné Description de la macro sélectionnée.
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Répétition Nombre de répétitions configuré.
Auto décl. Affiche la présence ou non d'une séquence de lancement automatique.
Créer Permet de lancer la création d'une nouvelle macro (voir Description de l'Interface d'Enregistrement de Macros).
Détruire Permet de détruire la macro sélectionnée (voir Remarques).
Démarrer Permet de lancer la macro sélectionnée.
Quitter Permet de fermer l'interface principale.

Remarques

Lorsque aucun fichier DynHelp n'est ouvert, les onglets Sommaire, Configuration, Rechercher et Index ne sont pas visibles.

En cas d'erreurs durant la création ou le lancement d'une macro, veuillez lancer l'application "DhpDgnst.exe" afin de réinitialiser l'état du système DynHelp avant de pouvoir le réutiliser (voir Description de l'Outil "DhpDgnst.exe").

Lorsqu'une interface principale DynHelp est ouverte sur une autre application, vous n'êtes pas autorisé à détruire ou à créer une macro. En effet, cela pourrait générer des erreurs si par exemple la macro que vous souhaitez détruire, est sélectionnée dans l'interface DynHelp d'une autre application.
Dans ce cas un message vous informe et vous propose de fermer cette ou ces dernières interfaces, avant de pouvoir réaliser ces opérations. Si toutefois ce message persiste alors qu'aucune interface DynHelp semble ouverte, veuillez lancer et utiliser l'outil "DhpDgnst.exe" pour initialiser l'état des applications utilisant le système DynHelp (voir Description de l'Outil "DhpDgnst.exe").

La suppression d'une macro ne se limite pas à détruire cette dernière du fichier DynHelp dans lequel elle se trouve, mais retire également celle-ci de la liste des macros favorites, y compris celles enregistrées dans le fichier et celles définies dans la séquence de lancement automatique.

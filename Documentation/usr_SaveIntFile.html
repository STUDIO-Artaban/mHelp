DynHelp: Guide Utilisateur V1.21
Interface d'Enregistrement: Fichier d'Enregistrement

L'étape "Fichier d'Enregistrement" de l'interface d'enregistrement, vous permet de sélectionner ou de créer le fichier DynHelp dans lequel sera sauvegardé la macro en cours de création.


Sujet Sujet du fichier DynHelp dans lequel vous pouvez sauvegarder la macro en cours de création.
Chemin du fichier Nom et chemin complet du fichier DynHelp dans lequel vous pouvez sauvegarder la macro en cours de création.
Liste des fichiers Liste des fichiers DynHelp dans lesquels vous pouvez sauvegarder la macro en cours de création.
... Permet d'enregistrer le nouveau fichier DynHelp, après avoir défini le sujet de ce fichier.
Nouveau Permet de lancer la création d'un nouveau fichier DynHelp, dans lequel vous pourrez sauvegarder la macro en cours de création. Cela a pour effet d'ajouter un élément dans la liste des fichiers DynHelp, afin de définir le sujet de ce nouveau fichier, avant de l'enregistrer via le bouton "..." décrit ci-dessus.
Ajouter Permet de charger un fichier DynHelp existant, afin d'y sauvegarder la macro en cours de création.
Retirer Permet de retirer de la liste des fichiers DynHelp, le fichier sélectionné.
Statut Statut du chargement du fichier DynHelp sélectionné, pour y sauvegarder la macro en cours de création.
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
‹ Précédent Permet de revenir à l'étape précédente: Liste des Contrôles (voir Interface d'Enregistrement: Liste des Contrôles).
Suivant › Permet de passer à l'étape suivante (voir Interface d'Enregistrement: Description de la Macro).
Terminer Permet de terminer le processus de création de macros.

Interface de Configuration: Onglet "Etat" de Type Booléen

L'onglet "Etat" de l'interface de configuration, vous permet de modifier la valeur du paramètre associé à l'état d'un contrôle, initialement fixée durant la création de la macro. Cette interface dépend du type de contrôle et donc du type de paramètre qui y est associé (voir Description des Contrôles et des Types de Paramètre). Dans l'exemple qui suit il s'agit d'un paramètre de type Booléen (B).

Contrôles de type: CheckBox, ToolBar checkbox


Commun Onglet "Commun" permettant de configurer le comportement à suivre dans le cas où le contrôle en cours de configuration, serait invisible ou inactif durant (voir Interface de Configuration: Onglet "Commun").
A Propos Onglet contenant des informations sur le contrôle sélectionné.
Activé Permet de modifier l'état initial du contrôle, de manière à cocher ce CheckBox lors du lancement de la macro.
Désactivé Permet de modifier l'état initial du contrôle, de manière à décocher ce CheckBox lors du lancement de la macro.
Change l'état/Indéterminé Permet de modifier l'état initial du contrôle, de manière à changer l'état de ce CheckBox lors du lancement de la macro / Permet de modifier l'état initial du contrôle, de manière à positionner ce CheckBox dans l'état indéterminé, lors du lancement de la macro (voir Remarques).
Valeur de l'état Valeur du paramètre (0: Désactivé, 1: Activé, 2: Change l'état/Indéterminé).
Description Permet de définir la description du paramètre (voir Remarques).
Type Affiche le type de paramètre (voir Description des Contrôles et des Types de Paramètre).
Limite Permet de définir la limite du paramètre, fixée par défaut à [0;2] pour ce type de contrôle (voir Remarques).
Aide (F1) Permet de lancer l'aide utilisateur du système DynHelp.
Appliquer Permet d'appliquer les changements configurés pour le contrôle sélectionné.
Fermer Permet de fermer l'interface.

Remarques

L'option "Change l'état/Indéterminé" est lié au fait qu'un contrôle de type CheckBox peut avoir suivant son style, deux états comme dans l'exemple (Activé/Désactivé), ou trois états (Activé/Désactivé/Indéterminé).

Les zones d'édition permettant de définir la description du paramètre et de ses limites, ne peuvent être éditées que dans le cas où vous souhaitez créer une macro paramétrable, et qu'il s'agit là de la configuration d'un de ses paramètres (voir Interface d'Enregistrement: Paramètres de la Macro).

S'il s'agit de la configuration d'un des paramètres de la future macro paramétrable, modifier l'état du contrôle revient à configurer la valeur par défaut du paramètre qui y est associé.

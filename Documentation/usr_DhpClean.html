Description de l'Outil "DhpClean.exe"

Cette application permet de réaliser la tâche ci-dessous, après une erreur ayant entraîné l'arrêt d'un des processus du système DynHelp de manière non conforme :

Réinitialiser l'état actuel du système DynHelp et de ses processus.

Lorsqu'un processus de création ou de lancement de macro est effectué, le système DynHelp stock des informations dans la base de registres assurant ainsi le partage de ces processus d'une application à une autre. L'intérêt de cet outil est de permettre à l'utilisateur d'initialiser ces clés dans le cas ou l'un de ces processus ne se termine pas correctement, après une erreur quelconque.

En effet, les opérations de création, de suppression et de lancement de macros ne peuvent être réalisées que sous certaines conditions, décrites ci-dessous :

Opérations Conditions nécessaires
Création d'une macro Pas de macro en cours de création.
Pas de macro en cours de lancement.
Pas d'interface principale DynHelp ouverte.
Destruction d'une macro Pas de macro en cours de création.
Pas de macro en cours de lancement.
Pas d'interface principale DynHelp ouverte.
Lancement d'une macro Pas de macro en cours de création.
Pas de macro en cours de lancement.

C'est pourquoi, sans cette initialisation le système DynHelp se comporte de manière non cohérente, les clés contenues dans la base de registres lui informant qu'il est toujours en cours de création ou en cours de lancement de macro.

DynHelp: Guide Utilisateur V1.21
Description de l'Interface Principale

L'interface principale de DynHelp comporte plusieurs onglets, vous permettant de réaliser les tâches principales décrites ci-dessous :

Ouverture et fermeture d'un ou plusieurs fichiers DynHelp.
Recherche, configuration et lancement d'une macro.
Destruction d'une macro.
Lancement de la création d'une nouvelle macro.
Gestion des macros favorites.

Remarques

Le message suivant est affiché lorsque vous souhaitez ouvrir l'interface principale DynHelp, alors qu'il y a une opération de création ou de lancement de macro en cours:



Si ce message persiste alors qu'il n'y a pas de création ou de lancement de macro en cours, veuillez utiliser l'outil "DhpDgnst.exe" afin de vous assurer que l'état actuel du système DynHelp est bien en attente (voir Description de l'Outil "DhpDgnst.exe").

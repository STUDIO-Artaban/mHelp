Introduction
Bienvenue dans le guide utilisateur du système d'aide et d'assistance DynHelp. Vous trouverez dans ce document toute l'aide dont vous avez besoin, afin d'utiliser au mieux cet outil. Chacune des interfaces du système DynHelp sont décrites minutieusement, vous permettant ainsi de profiter de toutes les innovations fournies par ce système.

Liste des fonctionnalités fournies par DynHelp :

Aide utilisateur
Système de macros
Système de répétitions de macros évolutives
Système de partage du processus d'aide et d'assistance d'un logiciel à un autre
Système d'accès aux fonctionnalités du logiciel

Chaque description des interfaces est interactive, autrement dit vous pouvez par un simple clique sur les figures représentant ces interfaces, obtenir la description de la fonction du contrôle sur lequel vous avez cliqué, ou passer à la description d'une autre interface appelée via ce même contrôle.

Chacune des remarques a son importance, car elles décrivent pour la plus part d'entre elles des comportements spécifiques liés à des cas particuliers, qui vous permettront de bien cerner le fonctionnement de cet outil.
Description
Ce guide utilisateur est divisé en deux parties : La première partie contient la description de l'interface principale (voir Description de l'Interface Principale), et la deuxième partie, la description de l'interface d'enregistrement de macros (voir Description de l'Interface d'Enregistrement de Macros).
